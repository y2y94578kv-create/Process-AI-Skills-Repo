### 6. 常见缺陷速查表

| 缺陷类型 | 外观特征 | 可能原因 | 处理措施 |
|----------|----------|----------|----------|
| **虚焊** | 焊点灰暗、无光泽、润湿不良 | 温度不足 / 焊盘氧化 / 助焊剂失效 | 调整温度 / 清洁焊盘 / 更换助焊剂 |
| **冷焊** | 焊点粗糙、颗粒状、强度低 | 工件移动 / 焊料未完全凝固 | 固定夹具 / 延长冷却时间 |
| **桥接** | 相邻焊点间有锡连接 | 锡量过多 / 焊盘间距过小 | 减少送丝量 / 优化焊盘设计 |
| **锡珠** | 焊点周围有散落锡粒 | 送丝过快 / 温度过高 / 助焊剂飞溅 | 降低送丝速度 / 降低温度 |
| **拉尖** | 烙铁撤离后焊点有尖刺 | 撤离速度过慢 / 温度偏低 | 调整撤离参数 / 提高温度 |
| **润湿不良** | 焊料未铺展，呈球状 | 焊盘氧化 / 助焊剂活性不足 | 清洁焊盘 / 更换助焊剂 |

### 4M1E 根因分类说明

| 分类 | 含义 | 典型根因举例 |
|------|------|-------------|
| **人 (Man)** | 人为操作因素 | 未校准、未按 SOP 执行、疲劳 |
| **机 (Machine)** | 设备因素 | PID 漂移、烙铁头氧化、送丝机构卡顿 |
| **料 (Material)** | 材料因素 | 锡线直径不均、助焊剂活性不足、焊盘氧化 |
| **法 (Method)** | 方法因素 | SOP 缺失、参数设定不当、工装设计缺陷 |
| **环 (Environment)** | 环境因素 | 温湿度超标、静电、灰尘 |
