# 🔧 工艺 7：锡线焊接工艺 (Wire Soldering)

---

## 📄 7.1 URS — 锡线焊接设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 自动化锡线焊接设备 |
| **设备节拍 (C/T)** | 单点 ≤ 3 秒 |
| **制程能力要求** | 焊点质量 **GR&R ≤ 10%** |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **烙铁头类型** | 恒温烙铁头（推荐：Weller / Hakko） |
| **烙铁温度** | 200-450°C 可调 |
| **温度精度** | ≤ ±3°C |
| **焊接时间** | 0.5-10 秒可调 |
| **锡线送丝速度** | 5-50 mm/s 可调 |
| **送丝精度** | ≤ ±0.5 mm |
| **助焊剂涂覆** | 自动助焊剂涂覆系统 |
| **视觉定位** | 高精度视觉定位，定位精度 ≤ ±0.1 mm |

### 3. 控制策略与防错逻辑 (软件要求)

1.  **恒温控制:** PID 温控，实时补偿热量损失，温度波动 ≤ ±3°C。
2.  **焊接时间监控:** 精确控制烙铁接触时间，防止过热。
3.  **送丝量控制:** 精确控制锡线送丝量，保证焊点大小一致。
4.  **焊点检测:** 焊接后视觉检测焊点外观（光泽、形状、润湿角）。
5.  **防虚焊:** 温度不足或时间过短时自动报警。

### 4. 工业控制、MES与数据追溯

| 项目 | 规格要求 |
|------|----------|
| **控制系统** | 工控机 + 运动控制卡 |
| **温控模块** | 高精度 PID 温控器 |
| **视觉系统** | Keyence / Cognex 视觉检测 |
| **数据上传** | OPC UA |
| **数据内容** | 产品ID、温度、时间、焊点图像、OK/NOK |
| **数据存储** | ≥50万条记录，支持图像追溯 |

### 5. EHS (环境、健康与安全)

| 项目 | 要求 |
|------|------|
| **排烟** | 焊接烟尘抽排装置（符合 GBZ 2.1） |
| **PPE** | 防化手套、安全眼镜、防尘口罩 |
| **防护罩** | 透明防护罩，联锁装置 |
| **烫伤防护** | 烙铁头防护罩，高温警示标识 |

### 6. 设备验收标准 (FAT / SAT)

| 验收阶段 | 方法 | 合格标准 |
|----------|------|----------|
| **预验收 (FAT)** | 供应商场地连续焊接 500 点 | 焊点外观合格率 ≥ 99%，拉力测试达标 |
| **终验收 (SAT)** | 买方产线，完成 GR&R 测试 | 连续运行 24 小时稳定 |

## 📄 7.2 SOP — 锡线焊接标准作业指导书

### 1. 准备工作与 5S 要求

| 项目 | 要求 |
|------|------|
| **PPE** | 防化手套、安全眼镜、防尘口罩 |
| **5S** | 工位整洁，工具定置定位 |
| **开班点检** | 检查烙铁温度、送丝机构、视觉系统 |
| **首件检验** | 焊接 10 个点，检查焊点外观和拉力 |

### 2. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 扫描产品条码，系统自动调用焊接程序 | 条码错误，程序锁定 |
| **2. 上料** | 将产品放入定位工装 | 定位准确，无偏移 |
| **3. 涂覆助焊剂** | 系统自动涂覆助焊剂 | 涂覆均匀，无遗漏 |
| **4. 焊接** | 系统自动执行焊接（烙铁接触 + 送丝） | 温度、时间、送丝量准确 |
| **5. 检测** | 视觉系统自动检测焊点外观 | 光亮、润湿良好、无虚焊 |
| **6. 判定** | 系统自动判定 OK/NOK | OK 件取出；NOK 件隔离 |
| **7. 清洁** | 定期清洁烙铁头 | 防止氧化 |

### 3. 异常处理与升级流程 (Escalation)

| 级别 | 触发条件 | 处理措施 |
|------|----------|----------|
| **轻微报警** | 单次 NOK | 复位重试，清洁烙铁头 |
| **严重报警** | 连续 3次 NOK，或温度异常 | **立即停止**，拉 Andon，通知工艺工程师 |

### 4. 首件检验规范

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验方法** | 焊接 10 个点，目视检查焊点外观，拉力测试 |
| **合格标准** | 焊点光亮、润湿良好，拉力 ≥ 设计值 |
| **记录** | 填写首件检验记录表，班长签字确认 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品焊接前必须扫码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与焊接数据关联上传 MES |

## 📄 7.3 PFMEA — 锡线焊接过程失效模式与影响分析

| 过程步骤 | 潜在失效模式 (FM) | 潜在失效影响 (Effect) | 严重度 (S) | 潜在失效原因 (Cause) | 频度 (O) | 现行预防控制 (PC) | 现行探测控制 (DC) | 探测度 (D) | RPN | 建议优化措施 (Action) |
|----------|-------------------|-----------------------|------------|-----------------------|----------|-------------------|-------------------|------------|-----|----------------------|
| **10. 温度控制** | 虚焊 | 强度不足，电气失效 | 8 | 温度不足，时间过短，焊盘氧化 | 3 | 温度监控，定期校准，表面清洁 | **焊点外观检测**，拉力抽检 | 3 | **72** | 引入焊点 X-Ray 检测 |
| **20. 焊接执行** | 冷焊 | 强度不足 | 7 | 工件移动，焊料未凝固 | 2 | 工件固定装置，凝固时间控制 | **焊点外观检测** | 3 | **42** | 引入振动隔离装置 |
| **20. 焊接执行** | 桥接 | 短路，功能异常 | 8 | 锡量过多，焊盘间距小 | 3 | 锡量控制，焊盘设计优化 | **视觉检测桥接**，异常报警 | 3 | **72** | 引入桥接自动修复功能 |
| **20. 焊接执行** | 锡珠 | 污染，短路风险 | 5 | 送丝过快，温度过高，助焊剂飞溅 | 3 | 送丝速度优化，温度控制 | **视觉检测锡珠**，异常报警 | 3 | **45** | 引入低飞溅助焊剂 |
