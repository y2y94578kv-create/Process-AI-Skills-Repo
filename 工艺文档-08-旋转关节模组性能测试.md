# 🔧 工艺 8：旋转关节模组性能测试 (Rotary Joint Module Performance Test)

---

## 📄 8.1 URS — 旋转关节模组测试设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 旋转关节模组性能测试设备 |
| **设备节拍 (C/T)** | 单件 ≤ 120 秒（含安装、测试、判定） |
| **制程能力要求** | 旋转精度 **Cpk ≥ 1.67**，测量系统 **GR&R ≤ 10%** |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **旋转精度** | ≤ ±0.05° |
| **旋转力矩** | 0.1-50 Nm 可测 |
| **转速范围** | 1-1000 rpm 可调 |
| **运行时间** | 1-3600 秒可调 |
| **温升限制** | ≤ 40°C |
| **角度编码器** | 高精度绝对值编码器（推荐：Heidenhain / Renishaw），精度 ≤ ±0.01° |
| **力矩传感器** | 高精度扭矩传感器（推荐：HBM / Kistler），精度 ≤ ±0.1% |
| **温度传感器** | 高精度温度传感器（Pt100），精度 ≤ ±0.1°C |

### 3. 控制策略与防错逻辑 (软件要求)

1.  **角度精度测试:** 编码器实时测量旋转角度，计算精度误差。
2.  **力矩稳定性测试:** 力矩传感器实时测量旋转力矩，计算波动范围。
3.  **温升监控:** 温度传感器实时监控模组温度，超限报警。
4.  **振动监控:** 振动传感器监控运行过程中的振动值。
5.  **自动报告生成:** 测试完成后自动生成测试报告。

### 4. 工业控制、MES与数据追溯

| 项目 | 规格要求 |
|------|----------|
| **控制系统** | 工控机 + 高速数据采集卡 |
| **数据采集** | 采样率 ≥ 10kHz |
| **数据上传** | OPC UA |
| **数据内容** | 产品ID、角度误差、力矩波动、温升、OK/NOK |
| **数据存储** | ≥50万条记录，支持趋势分析 |

### 5. EHS (环境、健康与安全)

| 项目 | 要求 |
|------|------|
| **防护罩** | 全封闭防护罩，联锁装置，测试过程中禁止开门 |
| **急停按钮** | 设备四角各一个 |
| **噪音等级** | ≤ 75 dB |
| **PPE** | 安全眼镜、劳保鞋 |

### 6. 设备验收标准 (FAT / SAT)

| 验收阶段 | 方法 | 合格标准 |
|----------|------|----------|
| **预验收 (FAT)** | 供应商场地使用标准模子验证 | 角度误差 ≤ ±0.05°，力矩波动 ≤ ±1% |
| **终验收 (SAT)** | 买方产线，完成 GR&R 测试 | 连续运行 24 小时，测量一致性 Cpk ≥ 1.67 |

## 📄 8.2 SOP — 旋转关节模组性能测试标准作业指导书

### 1. 准备工作与 5S 要求

| 项目 | 要求 |
|------|------|
| **PPE** | 安全眼镜、劳保鞋 |
| **5S** | 工位整洁，工具定置定位 |
| **开班点检** | 检查编码器、力矩传感器、温度传感器校准状态 |
| **首件检验** | 使用标准模子（已知精度）验证设备测量精度 |

### 2. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 扫描产品条码，系统自动调用测试程序 | 条码错误，程序锁定 |
| **2. 安装** | 将模组安装到测试台上，连接传感器 | 安装牢固，传感器连接正常 |
| **3. 启动** | 关闭防护门，按下启动按钮 | 防护门未关闭，设备无法启动 |
| **4. 测试** | 系统自动执行测试（精度、力矩、温升） | 观察测试过程，无异常 |
| **5. 判定** | 系统自动判定 OK/NOK | OK 件取出；NOK 件隔离 |
| **6. 记录** | 系统自动生成测试报告 | 数据自动上传 MES |

### 3. 异常处理与升级流程 (Escalation)

| 级别 | 触发条件 | 处理措施 |
|------|----------|----------|
| **轻微报警** | 单次 NOK | 复位重测，确认模组状态 |
| **严重报警** | 连续 3次 NOK，或设备异响/温升异常 | **立即停止**，按急停，拉 Andon，通知设备工程师 + 工艺工程师 |

### 4. 首件检验规范

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验工具** | 标准模子（已知精度） |
| **检验方法** | 安装标准模子进行测试，确认设备测量精度 |
| **合格标准** | 测量值与标准值偏差 ≤ ±5% |
| **记录** | 填写首件检验记录表，班长签字确认 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品测试前必须扫码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与测试数据关联上传 MES |
