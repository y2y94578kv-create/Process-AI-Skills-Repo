# 🔧 工艺 4：动平衡测试 (Dynamic Balancing Test)

---

## 📄 4.1 URS — 动平衡测试设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 自动动平衡测试与校正设备 |
| **设备节拍 (C/T)** | 单件 ≤ 30 秒（含测量、计算、配重、复测） |
| **制程能力要求** | 残余不平衡量 **Cpk ≥ 1.67**，测量系统 **GR&R ≤ 10%** |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **测试转速** | 500-10000 rpm 可调（变频调速） |
| **不平衡量检测精度** | ≤ ±0.1 g·mm |
| **平衡精度等级** | 支持 G0.4 ~ G6.3（ISO 1940） |
| **配重方式** | 自动打孔去重 / 自动加配重块 |
| **配重孔深度** | 0-30 mm 可调 |
| **配重位置精度** | ≤ ±1° |
| **传感器** | 高精度振动传感器（推荐：PCB / B&K） |

### 3. 控制策略与防错逻辑 (软件要求)

1.  **影响系数法:** 自动测量两个校正面的影响系数，计算不平衡量大小和相位。
2.  **自动配重计算:** 根据测量结果，自动计算配重孔位置和深度/配重块重量。
3.  **复测验证:** 配重后自动复测，确认残余不平衡量达标。
4.  **ISO 1940 报告:** 自动生成平衡报告，包含不平衡量、相位、精度等级、达标判定。

### 4. 工业控制、MES与数据追溯

| 项目 | 规格要求 |
|------|----------|
| **PLC 控制系统** | 西门子 S7-1200 或 Beckhoff |
| **数据采集** | 高速数据采集卡，采样率 ≥ 50kHz |
| **数据上传** | OPC UA |
| **数据内容** | 产品ID、不平衡量、相位、配重方案、OK/NOK |
| **数据存储** | ≥100万条记录，支持趋势分析 |

### 5. EHS (环境、健康与安全)

| 项目 | 要求 |
|------|------|
| **防护罩** | 全封闭防护罩，联锁装置，转速 >1000rpm 时禁止开门 |
| **急停按钮** | 设备四角各一个，触发后 0.5 秒内完全停止 |
| **噪音等级** | ≤ 80 dB（建议配备隔音房） |
| **PPE** | 安全眼镜、劳保鞋、耳塞（如噪音 >80 dB） |

### 6. 设备验收标准 (FAT / SAT)

| 验收阶段 | 方法 | 合格标准 |
|----------|------|----------|
| **预验收 (FAT)** | 供应商场地连续测试 300 件 | 残余不平衡量达标率 ≥ 99.5% |
| **终验收 (SAT)** | 买方产线，使用标准转子验证，完成 GR&R | 连续运行 24 小时，测量一致性 Cpk ≥ 1.67 |

## 📄 4.2 SOP — 动平衡测试标准作业指导书

### 1. 准备工作与 5S 要求

| 项目 | 要求 |
|------|------|
| **PPE** | 安全眼镜、劳保鞋、耳塞（如需） |
| **5S** | 工位整洁，工具定置定位 |
| **开班点检** | 检查振动传感器校准状态、防护罩联锁、转速稳定性 |
| **首件检验** | 使用标准转子（已知不平衡量）验证设备测量精度 |

### 2. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 扫描产品条码，系统自动调用平衡程序 | 条码错误，程序锁定 |
| **2. 安装** | 将转子安装到平衡机主轴上，确保同轴度 | 安装不到位，设备报警 |
| **3. 初始测量** | 启动设备，测量初始不平衡量和相位 | 系统自动记录数据 |
| **4. 配重计算** | 系统自动计算配重方案（位置、深度/重量） | 显示配重方案，等待确认 |
| **5. 配重校正** | 按计算结果进行打孔去重或加配重块 | 打孔深度/配重重量必须准确 |
| **6. 复测验证** | 重新测量残余不平衡量 | 残余不平衡量 ≤ 目标值 |
| **7. 判定** | 系统自动判定 OK/NOK | OK 件取出；NOK 件重新校正或隔离 |
| **8. 记录** | 系统自动生成平衡报告 | 数据自动上传 MES |

### 3. 异常处理与升级流程 (Escalation)

| 级别 | 触发条件 | 处理措施 |
|------|----------|----------|
| **轻微报警** | 复测不合格 | 重新配重校正，再次复测 |
| **严重报警** | 连续 3次复测不合格，或设备异响 | **立即停止**，按急停，拉 Andon，通知设备工程师 + 工艺工程师 |

### 4. 首件检验规范

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验工具** | 标准转子（已知不平衡量） |
| **检验方法** | 安装标准转子进行测量，确认设备测量精度 |
| **合格标准** | 测量值与标准值偏差 ≤ ±5% |
| **记录** | 填写首件检验记录表，班长签字确认 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品测试前必须扫码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与平衡数据关联上传 MES |

## 📄 4.3 PFMEA — 动平衡测试过程失效模式与影响分析

> 基于 AIAG-VDA 统一标准格式编制

| 过程步骤 | 潜在失效模式 (FM) | 潜在失效影响 (Effect) | 严重度 (S) | 潜在失效原因 (Cause) | 频度 (O) | 现行预防控制 (PC) | 现行探测控制 (DC) | 探测度 (D) | RPN | 建议优化措施 (Action) |
|----------|-------------------|-----------------------|------------|-----------------------|----------|-------------------|-------------------|------------|-----|----------------------|
| **10. 安装定位** | 安装同轴度偏差 | 测量数据不准确 | 6 | 主轴磨损，工装定位不准 | 2 | 定期校验主轴和工装 | 安装后自动检测同轴度 | 3 | **36** | 每季度用三坐标校验工装 |
| **20. 测量计算** | 不平衡量超标 | 振动过大，轴承寿命缩短 | 7 | 原始不平衡量过大，传感器漂移 | 3 | 传感器定期校准，标准转子验证 | **复测验证**，残余量超标报警 | 2 | **42** | 引入传感器状态在线监测 |
| **30. 配重校正** | 配重脱落 | 平衡失效，设备损坏 | 8 | 固定方式不可靠，振动松动 | 2 | 配重工艺优化，使用高强度胶 | 复测验证，配重后振动检测 | 3 | **48** | 引入配重固定力检测 |
| **40. 复测验证** | 复测数据不准确 | 误判 OK/NOK | 7 | 传感器故障，主轴跳动 | 2 | 传感器冗余设计，主轴定期维护 | 标准转子定期验证 | 2 | **28** | 增加标准转子验证频次 |

### RPN 优先级说明

| RPN 范围 | 优先级 | 处理要求 |
|----------|--------|----------|
