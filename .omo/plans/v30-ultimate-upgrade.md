## 二、V3.0 升级标准（每份文档必须包含）

### 2.1 URS 部分（设备技术规范 — 招标级）

| 要素 | V2.0 覆盖度 | V3.0 强制新增内容 |
|------|-------------|-------------------|
| 核心参数 | ✅ 基本参数 | 增加 **传感器精度、响应时间、量程、介质规格** 完整规格串 |
| 电气品牌清单 | ❌ 缺失 | PLC（西门子/三菱）+ 气动元件（SMC/Festo）+ 传感器（Keyence/Cognex）+ 安全器件（SICK/Pilz）+ 低压电气（施耐德） |
| 控制防错逻辑 | ⚠️ 简单描述 | 详细到 **Poka-Yoke 防呆逻辑点列表**（每条有触发条件 + 锁定行为） |
| MES 握手协议 | ⚠️ 提到 OPC UA | 完整 12 字段数据字典 + 上传失败重试机制 + 本地缓存策略 + 网络心跳监测 |
| EHS + 人体工程学 | ⚠️ 基本安全项 | 安全光栅响应时间 <10ms + 双手启动时间差 + 急停硬线回路 + 三色灯规范 + 噪音 + 上料高度 + 操作力限定 |
| FAT/SAT 量化指标 | ⚠️ 简单通过/不通过 | **Cpk ≥ 1.67** + **GR&R %SV ≤ 10%** + **OEE ≥ 95%** + **MTBF ≥ 5000h** + **MTTR ≤ 30min** + 验收计算公式 |

### 2.2 SOP 部分（标准作业指导书 — 车间受控级）

| 要素 | V2.0 覆盖度 | V3.0 强制新增内容 |
|------|-------------|-------------------|
| PPE | ⚠️ 列出 | 增加 **PPE 强制佩戴验证机制** |
| 班前点检 | ⚠️ 标准漏孔验证 | **高限件 / 低限件 双验证** + 安全光栅 + 急停按钮 + 压力清零 每日逐项检查 |
| 首件检验 | ⚠️ 提及 | 明确 OK/NOK 标准件制备规格 + **连续 3 次验证规则** + 不合格锁定机制 |
| 标准操作步骤 | ✅ 有 | **3 列表格**：动作拆解 → CTQ（质量关注点）→ 异常防呆 + 关键步骤 `[插入图片]` 占位符 |
| Escalation 矩阵 | ⚠️ 简单 2 级 | **4 级升级**：L1 操作工 → L2 班长 → L3 PE/QE → L4 工程经理 + **连续 3 件 NOK 强制停机红线** |
| 换型规范 | ❌ 缺失 | 换型步骤 + 目标换型时间 ≤ 10 min |

### 2.3 PFMEA 部分（过程失效模式分析 — AIAG-VDA 标准）

| 要素 | V2.0 覆盖度 | V3.0 强制新增内容 |
|------|-------------|-------------------|
| PC vs DC 区分 | ❌ 混用 | **现行预防控制(PC)** 必须是防呆设计/参数硬约束；**现行探测控制(DC)** 必须是传感器/视觉检测 — 严格独立列，内容不能交换 |
| 根因分析 4M1E | ❌ 简单原因 | 每个 FM 原因分析须标注 **4M1E 标签**（人-机-料-法-环），深入到物理/化学根因 |
| S/O/D 评分 | ✅ 有 | 保持，增加一致性评审 |
| 行动建议(Action) | ⚠️ 笼统 | **具体工程改善** + **责任方** + **行动优先级(AP: H/M/L)** |
| RPN 阈值 | ✅ 有 | 增加 **AP 优先级判定矩阵**（高/中/低 处理时限） |

## 四、V3.0 每份文档详细内容模板

### 4.1 URS 结构（约 80-120 行/份）

```
## URS — [工艺] 设备技术规范（设备招标级）

### 1. 项目概况与总体要求
  - T/T 节拍、OEE/MTBF/MTTR 目标、Cpk/GR&R 指标

### 2. 核心工艺技术参数（硬件要求）
  - [工艺专用参数表] + 传感器品牌精度规格

### 3. 控制与电气系统要求
  #### 3.1 PLC 与核心电气元件品牌清单
    - 9 类元器件 × 2 方案（西门子/三菱 + SMC/Festo + Keyence/Cognex + SICK/Pilz + 施耐德）
  #### 3.2 控制与防错逻辑要求
    - 4-6 条具体 Poka-Yoke 防呆逻辑点

### 4. MES / 信息化握手协议
    - OPC UA 协议、12 字段数据字典、上传重试机制、本地缓存策略

### 5. EHS 与人体工程学要求
    - 安全光栅/双手启动/急停/三色灯/噪音/上料高度/操作力

### 6. 设备验收量化指标（FAT / SAT）
    - Cpk≥1.67 / GR&R≤10% / OEE≥95% / MTBF≥5000h 验收公式
```

### 4.2 SOP 结构（约 80-100 行/份）

```
## SOP — [工艺] 标准作业指导书（车间受控级）

### 1. 开班准备与防呆验证
  #### 1.1 每日开班点检清单（6-8 项逐行检查表）
  #### 1.2 首件检验规程（OK/NOK 标准件 + 3/3 验证 + 不合格锁定）

### 2. 标准操作步骤（动作拆解 → CTQ → 防呆）
    - 6-8 步 × 3 列表格 + [插入图片] 占位符

### 3. 异常处理与升级流程（Escalation Matrix）
    - L1/L2/L3/L4 四级 + 连续 3 件 NOK 强制停机红线

### 4. 换型操作规范
    - 6 步换型流程 + 换型时间 ≤ 10 min
```
