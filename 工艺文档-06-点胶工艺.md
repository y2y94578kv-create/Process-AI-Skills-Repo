# 🔧 工艺 6：点胶工艺 (Dispensing/Gluing)

---

## 📄 6.1 URS — 点胶设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 高精度自动点胶机 |
| **设备节拍 (C/T)** | 单点 ≤ 2 秒 |
| **制程能力要求** | 胶点尺寸 **Cpk ≥ 1.33**，位置精度 **GR&R ≤ 10%** |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **点胶头类型** | 时间-压力阀 / 螺杆泵 / 容积式点胶阀（根据胶水特性选择） |
| **点胶量** | 0.01-5.0 ml 可调 |
| **点胶速度** | 10-500 mm/s 可调 |
| **点胶高度** | 0.5-20 mm 可调 |
| **点胶精度** | ≤ ±0.1 mm |
| **胶水温度** | 25-80°C 可调（恒温控制） |
| **视觉定位** | 高精度视觉定位系统（推荐：Keyence / Cognex），定位精度 ≤ ±0.05 mm |

### 3. 控制策略与防错逻辑 (软件要求)

1.  **时间-压力控制:** 精确控制施压时间和压力，保证点胶一致性。
2.  **体积计量:** 螺杆泵/活塞泵精确计量胶水体积。
3.  **视觉检测:** 点胶后视觉检测胶点尺寸、位置，不合格立即报警。
4.  **胶水粘度补偿:** 自动温度-粘度补偿，保证点胶量稳定。
5.  **防拉丝:** 点胶头快速回抽，防止拉丝。

### 4. 工业控制、MES与数据追溯

| 项目 | 规格要求 |
|------|----------|
| **控制系统** | 工控机 + 运动控制卡（推荐：PMAC / Galil） |
| **视觉系统** | Keyence / Cognex 视觉检测 |
| **数据上传** | OPC UA |
| **数据内容** | 产品ID、胶点尺寸、位置、OK/NOK |
| **数据存储** | ≥50万条记录，支持图像追溯 |

### 5. EHS (环境、健康与安全)

| 项目 | 要求 |
|------|------|
| **通风** | 局部排风装置，符合 GBZ 2.1 车间空气中有害物质限值 |
| **PPE** | 防化手套、安全眼镜、防尘口罩（如胶水有挥发性） |
| **防护罩** | 透明防护罩，联锁装置 |
| **急停按钮** | 设备四角各一个 |

### 6. 设备验收标准 (FAT / SAT)

| 验收阶段 | 方法 | 合格标准 |
|----------|------|----------|
| **预验收 (FAT)** | 供应商场地连续点胶 500 点 | 胶点尺寸误差 ≤ ±0.2mm，位置误差 ≤ ±0.3mm |
| **终验收 (SAT)** | 买方产线，完成 GR&R 测试 | 连续运行 24 小时，胶点一致性 Cpk ≥ 1.33 |

## 📄 6.2 SOP — 点胶标准作业指导书

### 1. 准备工作与 5S 要求

| 项目 | 要求 |
|------|------|
| **PPE** | 防化手套、安全眼镜、防尘口罩 |
| **5S** | 工位整洁，工具定置定位 |
| **开班点检** | 检查胶水状态（无气泡、无固化）、点胶头清洁、视觉系统校准 |
| **首件检验** | 点胶 10 个点，测量胶点尺寸和位置，确认合格 |

### 2. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 扫描产品条码，系统自动调用点胶程序 | 条码错误，程序锁定 |
| **2. 上料** | 将产品放入定位工装 | 定位准确，无偏移 |
| **3. 启动** | 关闭防护门，按下启动按钮 | 防护门未关闭，设备无法启动 |
| **4. 点胶** | 系统自动执行点胶路径 | 观察点胶过程，无拉丝、断胶 |
| **5. 检测** | 视觉系统自动检测胶点 | 尺寸、位置合格 |
| **6. 判定** | 系统自动判定 OK/NOK | OK 件取出；NOK 件隔离 |
| **7. 清洁** | 定期清洁点胶头 | 防止堵头 |

### 3. 异常处理与升级流程 (Escalation)

| 级别 | 触发条件 | 处理措施 |
|------|----------|----------|
| **轻微报警** | 单次 NOK（胶点尺寸偏差） | 复位重试，清洁点胶头 |
| **严重报警** | 连续 3次 NOK，或点胶头堵塞 | **立即停止**，拉 Andon，通知工艺工程师 |

### 4. 首件检验规范

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验方法** | 点胶 10 个点，使用视觉系统或千分尺测量胶点尺寸 |
| **合格标准** | 胶点尺寸误差 ≤ ±0.2mm，位置误差 ≤ ±0.3mm |
| **记录** | 填写首件检验记录表，班长签字确认 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品点胶前必须扫码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与点胶数据关联上传 MES |

## 📄 6.3 PFMEA — 点胶过程失效模式与影响分析

> 基于 AIAG-VDA 统一标准格式编制

| 过程步骤 | 潜在失效模式 (FM) | 潜在失效影响 (Effect) | 严重度 (S) | 潜在失效原因 (Cause) | 频度 (O) | 现行预防控制 (PC) | 现行探测控制 (DC) | 探测度 (D) | RPN | 建议优化措施 (Action) |
|----------|-------------------|-----------------------|------------|-----------------------|----------|-------------------|-------------------|------------|-----|----------------------|
| **10. 胶水准备** | 胶水有气泡 | 点胶量不准确 | 5 | 胶水静置时间不足，脱泡不充分 | 3 | 标准化脱泡 SOP，真空脱泡 | 首件点胶检测 | 3 | **45** | 引入自动脱泡装置 |
| **20. 点胶执行** | 拉丝 | 外观不良，污染其他区域 | 5 | 粘度过高，抬升速度过慢，温度过低 | 4 | 温度控制，抬升速度优化 | **视觉检测拉丝**，异常报警 | 3 | **60** | 引入防拉丝回抽功能 |
| **20. 点胶执行** | 溢胶 | 功能受影响，污染部件 | 6 | 点胶量过大，流动性过强，表面油污 | 3 | 点胶量参数优化，表面清洁工序 | **视觉检测溢胶**，异常报警 | 3 | **54** | 引入胶量重量检测 |
| **20. 点胶执行** | 少胶/断胶 | 粘接强度不足，产品脱落 | 8 | 供胶不足，堵头，管路气泡 | 3 | 供胶系统检查，定期清洁点胶头 | **视觉检测胶点缺失**，异常报警 | 3 | **72** | 引入供胶压力监控 |
