## 6.1 URS — 点胶设备技术规格书 (招标级)

### 1. 项目概况与总体要求

| 项目 | 规格要求 |
|------|----------|
| **设备名称** | 高精度自动点胶机 |
| **设备节拍 (C/T)** | 单点 ≤ 2 秒，多点 ≤ 4 秒 |
| **制程能力要求** | 胶点尺寸 **Cpk ≥ 1.33**，位置精度 **GR&R ≤ 10%**，**ndc ≥ 5** |
| **OEE 要求** | ≥ 95% |
| **MTBF** | ≥ 5000 小时 |
| **MTTR** | ≤ 30 分钟 |

### 2. 核心工艺技术参数

| 参数 | 规格要求 |
|------|----------|
| **点胶头类型** | 时间-压力阀 / 螺杆泵 / 容积式点胶阀（根据胶水特性选择） |
| **点胶量** | 0.01-5.0 ml 可调，分辨率 ≤ 0.001 ml |
| **点胶速度** | 10-500 mm/s 可调 |
| **点胶高度** | 0.5-20 mm 可调 |
| **点胶精度** | ≤ ±0.1 mm |
| **胶水温度** | 25-80°C 可调，恒温控制精度 ±0.5°C |
| **视觉定位** | 高精度视觉定位系统，定位精度 ≤ ±0.05 mm |
| **防拉丝** | 快速回抽功能，回抽量 0.005-0.05 ml 可调 |

### 3. 控制与电气品牌清单 (不可省略)

| 类别 | 品牌/型号 | 技术要求 | 备注 |
|------|----------|----------|------|
| **PLC** | 西门子 S7-1511(F) / 三菱 FX5U-32MT | 安全型 CPU，支持 PROFINET/EtherNet/IP | 主控制器 |
| **气动元件** | SMC / Festo | 电磁阀响应 ≤ 10ms，气缸重复定位精度 ±0.02mm | 含电磁阀、气缸、调压阀 |
| **传感器** | Keyence GP-M / Cognex | 视觉分辨率 ≥ 500 万像素，帧率 ≥ 30fps | 视觉传感器 + 光纤传感器 |
| **安全光栅** | SICK deTec4 Core / Keyence GL-R | 响应时间 ≤ 10ms，保护高度 ≥ 500mm | 覆盖上料区域 |
| **安全继电器** | Pilz PNOZ / SICK RB12 | 双通道监控，强制导向触点 | 安全回路 |
| **三色报警灯** | Patlite / Murrelektronik | 红/黄/绿 + 蜂鸣器，音量可调 | 设备状态指示 |
| **低压电气** | 施耐德 / 魏德米勒 | 断路器、接触器、端子，IP20 以上 | 配电柜内 |

### 4. 控制策略与防错逻辑

#### 4.1 核心控制功能

| 功能 | 描述 | 精度要求 |
|------|------|----------|
| **时间-压力控制** | 精确控制施压时间和压力，保证点胶一致性 | 压力控制精度 ±0.5 kPa |
| **体积计量** | 螺杆泵/活塞泵精确计量胶水体积 | 计量精度 ±1% |
| **视觉检测** | 点胶后视觉检测胶点尺寸、位置，不合格立即报警 | 检测精度 ±0.02mm |
| **胶水粘度补偿** | 自动温度-粘度补偿，保证点胶量稳定 | 温度补偿范围 25-80°C |
| **防拉丝** | 点胶头快速回抽，防止拉丝 | 回抽量精度 ±0.001 ml |
| **断胶检测** | 实时监控供胶压力，断胶自动报警 | 响应时间 ≤ 50ms |

#### 4.2 防错逻辑 (Poka-Yoke)

| 故障场景 | 防错措施 | 响应动作 |
|----------|----------|----------|
| 胶水温度超限 | 温度传感器实时监控 | 超限 ±2°C → 自动停机 |
| 点胶压力异常 | 压力传感器实时监控 | 异常 → 报警并锁机 |
| 视觉检测 NOK | 视觉系统自动判定 | NOK → 强制复检 3 次 |
| 安全光栅触发 | 光栅实时检测 | 触发 → 立即停机 |
| 防护门未关闭 | 门开关传感器 | 未关 → 设备不启动 |
| 双手启动不同步 | 双手按钮时间差检测 | 时间差 > 0.5s → 不启动 |
| 胶水余量不足 | 液位传感器监控 | 余量 < 10% → 报警提示 |
| 点胶头堵塞 | 供胶压力监控 | 压力异常 → 报警停机 |

## 6.2 SOP — 点胶标准作业指导书 (车间受控级)

### 1. 开班点检清单 (7 项)

| 序号 | 点检项目 | 方法 | 合格标准 | 不合格处理 | 记录 |
|------|----------|------|----------|------------|------|
| 1 | 胶水状态检查 | 目视 + 抽样 | 无气泡、无固化、粘度正常 | 更换胶水，重新脱泡 | ✓ |
| 2 | 点胶头清洁 | 目视 + 试胶 | 出胶顺畅，无堵塞 | 清洁/更换点胶头 | ✓ |
| 3 | 视觉系统校准 | 标准件校准 | 定位误差 ≤ ±0.05mm | 重新校准 | ✓ |
| 4 | 标准点胶验证 10 点 | 连续点胶 10 点 | 尺寸误差 ≤ ±0.2mm | 调整参数 | ✓ |
| 5 | 安全光栅测试 | 遮挡测试 | 响应时间 ≤ 20ms | 通知维修 | ✓ |
| 6 | 急停按钮测试 | 按下测试 | 设备立即停止 | 通知维修 | ✓ |
| 7 | 胶水温度确认 | 温度计测量 | 设定值 ± 0.5°C | 调整温控器 | ✓ |

#### 点检流程

```
开班 → 穿戴PPE → 开机预热(10min) → 执行7项点检 → 记录结果
                                              ↓
                                    全部合格 → 班长签字 → 首件检验 → 正常生产
                                              ↓
                                    任何不合格 → 立即处理 → 复检 → 合格后继续
```

### 2. 首件检验

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验方法** | 标准件点胶 10 点 → 视觉检测胶点尺寸/位置 × 3 次 |
| **合格标准** | 胶点尺寸误差 ≤ ±0.2mm，位置误差 ≤ ±0.3mm |
| **记录** | 填写首件检验记录表，班长签字确认 |

#### 首件检验流程

```
标准件上料 → 点胶10点 → 视觉检测 → 记录数据
                              ↓

                              ↓
            OK → 班长签字 → 正常生产
            NOK → 停机调整 → 重新首件 → 直到OK
```

### 3. 标准操作步骤 (6 步 × 3 列)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 扫描产品条码，系统自动调用点胶程序 | 条码错误 → 程序锁定，无法启动 |
| **2. 上料** | 将产品放入定位工装，确认定位销到位 | 定位偏移 → 视觉识别报警 |
| **3. 启动** | 关闭防护门，双手同时按下启动按钮 | 防护门未关/单手启动 → 设备不响应 |
| **4. 点胶** | 系统自动执行点胶路径 | 观察点胶过程，无拉丝、断胶 |
| **5. 检测** | 视觉系统自动检测胶点 | 尺寸/位置 NOK → 强制复检 |
| **6. 判定** | 系统自动判定 OK/NOK | OK → 取出；NOK → 隔离并记录 |

> 基于 AIAG-VDA 统一标准格式编制

|----------|-------------------|-----------------------|---|-----------------------|---|-------------------|-------------------|---|-----|----|------------------|--------|

### RPN / AP 优先级说明

| 行动优先级 (AP) | RPN 范围 | 处理要求 |
|----------------|----------|----------|
| **H (High)** | RPN ≥ 70 或 S ≥ 8 | 必须立即采取纠正措施，7 天内关闭 |
| **M (Medium)** | 40 ≤ RPN < 70 | 应制定改进计划，30 天内完成 |
| **L (Low)** | RPN < 40 | 维持现有控制，季度回顾 |

### 4M1E 根因分析总结

| 根因类别 | 点胶工艺常见问题 |
|----------|------------------|
| **Man (人)** | 脱泡操作不规范、参数设定错误、清洁不到位 |
| **Machine (机)** | 真空泵故障、回抽功能失效、堵头/管路泄漏、UV 灯功率衰减 |
| **Material (料)** | 胶水批次差异、粘度波动、固化特性变化 |
| **Method (法)** | 脱泡时间不足、点胶路径规划不当、固化参数设定不当 |
| **Environment (环)** | 车间温湿度异常、光照变化、表面油污 |
