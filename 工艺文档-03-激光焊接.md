## 📄 3.1 URS — 激光焊接设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 全自动激光焊接系统 |
| **设备节拍 (C/T)** | 单焊缝 ≤ 5 秒（含定位、焊接、冷却） |
| **制程能力要求** | 熔深 **Cpk ≥ 1.67**，焊接强度 **GR&R ≤ 10%** |
| **激光安全等级** | Class 1（全封闭防护罩） |
| **设计寿命** | ≥ 10 年 / 5000 万次循环 |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **激光器类型** | 光纤激光器（推荐：IPG / Trumpf / Rofin） |
| **激光功率** | 500-6000W 可调，功率波动 ≤±1% |
| **焊接速度** | 10-200 mm/s 可调，速度精度 ≤±0.5% |
| **离焦量** | -5 ~ +5 mm 可调，精度 ≤±0.1mm |
| **保护气** | 氩气/氮气，流量 5-30 L/min，精度 ≤±2% |
| **光学系统** | 自动对焦，焦点实时追踪，焦点直径 0.1-2.0mm |
| **运动系统** | 六轴机器人或三维龙门（推荐：FANUC / KUKA / ABB） |
| **定位精度** | ≤±0.05mm |
| **重复定位精度** | ≤±0.02mm |

### 3. 控制与电气品牌清单（不可省略）

| 类别 | 品牌/型号 |
|------|-----------|
| **PLC 控制系统** | 西门子 S7-1511(F) / 三菱 FX5U-32MT |
| **气动元件** | SMC / Festo |
| **传感器** | Keyence GP-M / Cognex |
| **安全光栅** | SICK deTec4 Core / Keyence GL-R（响应 ≤10ms） |
| **安全继电器** | Pilz PNOZ / SICK RB12 |
| **三色报警灯** | Patlite / Murrelektronik |
| **低压电气** | 施耐德 / 魏德米勒 |
| **机器人控制器** | FANUC R-30iB / KRC4 |
| **视觉系统** | Keyence / Cognex 焊缝跟踪系统 |

### 4. 控制策略与防错逻辑 (软件要求)

1.  **焦点位置实时追踪:** 激光头配备距离传感器，实时调整焦点位置，确保焊接质量稳定。
2.  **熔池形貌监控:** 高速相机实时拍摄焊接熔池，AI 算法分析熔池宽度、稳定性，异常时立即停止。
3.  **焊缝轨迹跟踪:** 激光视觉传感器实时跟踪焊缝位置，补偿工件装配误差。
4.  **能量监控:** 实时监控激光功率稳定性，功率波动 >±5% 时报警。
5.  **焊缝质量在线判定:** 基于熔池图像的 AI 算法，实时判定焊缝质量 OK/NOK。
6.  **参数密码锁定:** 焊接参数分级管理，操作工仅可调用，不可修改。

### 5. MES 信息化握手协议（OPC UA 强制）

| 上传字段 | 说明 |
|----------|------|

| **焊缝轨迹文件** | 焊缝轨迹坐标文件 |
| **保护气流量** | 实际保护气流量 (L/min) |
| **设备 ID** | 设备唯一标识 |

**通信要求：**

- 上传失败重试：3 次
- 本地缓存：SQLite，≥10000 条
- 数据存储：≥50万条记录，支持图像追溯

## 📄 3.2 SOP — 激光焊接标准作业指导书

### 1. 开班点检清单（6-8项）

| 序号 | 点检项目 | 点检方法 | 合格标准 | 责任人 |
|------|----------|----------|----------|--------|
| 1 | **激光功率稳定性验证** | 功率计测量，连续 10 次 | 功率波动 ≤±1% | 操作工 |
| 2 | **保护气流量校验** | 流量计读数 | 流量 5-30 L/min，精度 ≤±2% | 操作工 |
| 3 | **安全联锁测试** | 打开防护门，设备应立即停止 | 联锁响应 ≤10ms | 操作工 |
| 4 | **冷却水温检查** | 温度计读数 | 水温 20±2℃ | 操作工 |
| 5 | **标准试片焊接+金相检验** | 焊接标准试片，金相切片 | 熔深 ≥ 0.5mm，熔宽 1.0-3.0mm | 操作工+班长 |
| 6 | **紧急停止按钮测试** | 按下急停按钮 | 设备 0.5s 内完全停止 | 操作工 |
| 7 | **防护罩完整性检查** | 目视检查 | 无破损、无松动 | 操作工 |
| 8 | **焊丝/保护气余量检查** | 目视检查 | 余量充足 | 操作工 |

### 2. 首件检验

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验工具** | 标准试片、金相显微镜、千分尺 |
| **检验方法** | 焊接标准试片 → 金相切片检验熔深/熔宽 × 3次 |
| **合格标准** | 熔深 ≥ 0.5mm，熔宽 1.0-3.0mm，3次全部合格 |
| **记录** | 填写首件检验记录表，班长签字确认 |
| **不合格处理** | 首件不合格，禁止生产，重新调试并再次首件检验 |

### 3. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) | 防呆措施 |
|------|----------|-------------------|----------|
| **1. 扫码** | 扫描工件条码，系统自动调用焊接程序 | 条码与程序匹配 | 条码错误，程序锁定，禁止焊接 `[插入图片：扫码操作]` |
| **2. 清洁** | 使用无水酒精清洁焊接区域 | 表面无油污、无氧化物 | 清洁后 30s 内必须焊接 `[插入图片：清洁操作]` |
| **3. 装夹** | 将工件放入夹具，确保定位准确 | 工件间隙 ≤ 0.1mm | 间隙超差，夹具传感器报警 `[插入图片：装夹操作]` |
| **4. 启动** | 关闭防护门，双手启动 | 防护门关闭，双手同时按下 | 防护门未关闭，设备无法启动 `[插入图片：启动操作]` |
| **5. 监控** | 观察焊接过程监控画面 | 熔池稳定、无飞溅、无异常 | AI 熔池监控异常自动停止 `[插入图片：监控画面]` |
| **6. 判定** | 系统自动判定 OK/NOK | 焊缝质量合格 | OK 件取出；NOK 件隔离并标记 `[插入图片：判定操作]` |

### 4. Escalation 矩阵（4级）

| 级别 | 触发条件 | 处理措施 | 响应时间 | 责任人 |
|------|----------|----------|----------|--------|
| **L1-操作工** | 单件 NOK | 复位重焊，记录原因 | 2min 内处理 | 操作工 |
| **L2-班长** | **连续 3 件 NOK** | **强制停机**，通知设备工程师 | 5min 响应 | 班长 |
| **L3-PE/QE** | 连续 5 件 NOK | 工艺/质量工程师到场分析 | 30min 到场 | PE/QE |
| **L4-工程经理** | 批量 ≥20 件 NOK | 停线 + CAPA，启动 8D 报告 | 立即响应 | 工程经理 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品焊接前必须扫码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与焊接数据、熔池图像关联上传 MES |
| **数据字段** | SN、焊接时间、判定结果、激光功率、焊接速度、熔深、熔池图像路径、焊缝轨迹文件、保护气流量、设备 ID、操作工 ID、焊接次数计数 |
