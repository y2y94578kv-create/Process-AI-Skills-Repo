# 🔧 工艺 3：激光焊接 (Laser Welding)

---

## 📄 3.1 URS — 激光焊接设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 自动化激光焊接系统 |
| **设备节拍 (C/T)** | 单焊缝 ≤ 5 秒（含定位、焊接、冷却） |
| **制程能力要求** | 熔深 **Cpk ≥ 1.67**，焊接强度 **GR&R ≤ 10%** |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **激光器类型** | 光纤激光器（推荐：IPG / Trumpf / Rofin） |
| **激光功率** | 500-6000W 可调 |
| **焊接速度** | 10-200 mm/s 可调 |
| **离焦量** | -5 ~ +5 mm 可调 |
| **保护气** | 氩气/氮气，流量 5-30 L/min |
| **光学系统** | 自动对焦，焦点实时追踪 |
| **运动系统** | 六轴机器人或三维龙门（推荐：FANUC / KUKA / ABB） |

### 3. 控制策略与防错逻辑 (软件要求)

1.  **焦点位置实时追踪:** 激光头配备距离传感器，实时调整焦点位置，确保焊接质量稳定。
2.  **熔池形貌监控:** 高速相机实时拍摄焊接熔池，AI 算法分析熔池宽度、稳定性，异常时立即停止。
3.  **焊缝轨迹跟踪:** 激光视觉传感器实时跟踪焊缝位置，补偿工件装配误差。
4.  **能量监控:** 实时监控激光功率稳定性，功率波动 >±5% 时报警。

### 4. 工业控制、MES与数据追溯

| 项目 | 规格要求 |
|------|----------|
| **PLC 控制系统** | 西门子 S7-1500 或 Beckhoff TwinCAT |
| **机器人控制器** | FANUC R-30iB / KRC4 |
| **视觉系统** | Keyence / Cognex 焊缝跟踪系统 |
| **数据上传** | OPC UA |
| **数据内容** | 焊缝ID、功率、速度、熔深、OK/NOK、熔池图像 |
| **数据存储** | ≥50万条记录，支持图像追溯 |

### 5. EHS (环境、健康与安全)

| 项目 | 要求 |
|------|------|
| **激光防护** | 全封闭防护罩，激光安全联锁（Class 1 激光安全等级） |
| **护目镜** | 操作人员必须佩戴对应波长的激光防护眼镜（OD ≥ 5） |
| **排烟系统** | 焊接烟尘抽排装置，符合 GBZ 2.1 车间空气中有害物质限值 |
| **急停按钮** | 设备四角各一个，触发后 0.5 秒内完全停止 |
| **噪音等级** | ≤ 70 dB |

### 6. 设备验收标准 (FAT / SAT)

| 验收阶段 | 方法 | 合格标准 |
|----------|------|----------|
| **预验收 (FAT)** | 供应商场地连续焊接 500 条焊缝 | 金相切片合格率 ≥ 99.5%，拉拔力 ≥ 设计值 |
| **终验收 (SAT)** | 买方产线，提供功率标定证书，完成 GR&R | 连续运行 48 小时，焊缝一致性 Cpk ≥ 1.67 |

## 📄 3.2 SOP — 激光焊接标准作业指导书

### 1. 准备工作与 5S 要求

| 项目 | 要求 |
|------|------|
| **PPE** | 激光防护眼镜（OD≥5）、防割手套、劳保鞋、防尘口罩 |
| **5S** | 工位整洁，工具定置定位 |
| **开班点检** | 检查激光器功率稳定性、保护气流量、冷却水温、防护罩联锁 |
| **首件检验** | 使用标准试片进行焊接，进行金相切片检验熔深/熔宽 |

### 2. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 扫描工件条码，系统自动调用焊接程序 | 条码错误，程序锁定，禁止焊接 |
| **2. 清洁** | 使用无水酒精清洁焊接区域 | 去除油污、氧化物，确保表面清洁 |
| **3. 装夹** | 将工件放入夹具，确保定位准确 | 工件间隙 ≤ 0.1mm，否则报警 |
| **4. 启动** | 关闭防护门，按下启动按钮 | 防护门未关闭，设备无法启动 |
| **5. 监控** | 观察焊接过程监控画面 | 熔池稳定、无飞溅、无异常 |
| **6. 判定** | 系统自动判定 OK/NOK | OK 件取出；NOK 件隔离 |
| **7. 记录** | 系统自动记录焊接参数 | 数据自动上传 MES |

### 3. 异常处理与升级流程 (Escalation)

| 级别 | 触发条件 | 处理措施 |
|------|----------|----------|
| **轻微报警** | 单次 NOK（功率波动） | 复位重焊，记录原因 |
| **严重报警** | 连续 3次 NOK，或防护门联锁故障 | **立即停止**，按急停，拉 Andon，通知设备工程师 + 工艺工程师 |
| **激光安全事件** | 激光意外出光、防护罩打开 | 立即切断激光电源，疏散人员，通知 EHS 部门 |

### 4. 首件检验规范

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验工具** | 标准试片、金相显微镜 |
| **检验方法** | 焊接标准试片，进行金相切片，测量熔深/熔宽 |
| **合格标准** | 熔深 ≥ 0.5mm，熔宽 1.0-3.0mm |
| **记录** | 填写首件检验记录表，班长签字确认 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品焊接前必须扫码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与焊接数据、熔池图像关联上传 MES |
