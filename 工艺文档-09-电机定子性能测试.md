# 🔧 工艺 9：电机定子性能测试 (Motor Stator Performance Test)

---

## 📄 9.1 URS — 电机定子性能测试设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 电机定子综合性能测试设备 |
| **设备节拍 (C/T)** | 单件 ≤ 30 秒（含安装、测试、判定） |
| **制程能力要求** | 测量系统 **GR&R ≤ 10%** |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **绝缘电阻测试** | 测试电压 500V DC，测量范围 0-1000MΩ，精度 ≤ ±5% |
| **耐压测试** | 测试电压 0-5000V AC 可调，漏电流 ≤ 10mA |
| **匝间绝缘测试** | 冲击电压 0-5000V，波形比较分析 |
| **电感测试** | 测试频率 1kHz/10kHz，测量范围 0.1-1000mH，精度 ≤ ±1% |
| **接地电阻测试** | 测试电流 10A，测量范围 0-1Ω，精度 ≤ ±1% |

### 3. 控制策略与防错逻辑 (软件要求)

1.  **绝缘电阻测试:** 施加 500V DC，测量绕组与铁芯之间的绝缘电阻，判断是否 ≥ 100MΩ。
2.  **耐压测试:** 施加 1500V AC/1min，监测漏电流，判断是否击穿。
3.  **匝间绝缘测试:** 施加冲击电压，比较各相波形，判断是否有匝间短路。
4.  **电感测试:** 测量各相电感值，判断是否在设计值 ±10% 范围内。
5.  **自动判定:** 系统自动判定所有测试项，生成测试报告。

### 4. 工业控制、MES与数据追溯

| 项目 | 规格要求 |
|------|----------|
| **控制系统** | 工控机 + 高精度数据采集卡 |
| **测试模块** | 高精度绝缘电阻测试仪、耐压测试仪、匝间绝缘测试仪、LCR 表 |
| **数据上传** | OPC UA |
| **数据内容** | 产品ID、绝缘电阻、耐压、匝间绝缘、电感、OK/NOK |
| **数据存储** | ≥100万条记录，支持趋势分析 |

### 5. EHS (环境、健康与安全)

| 项目 | 要求 |
|------|------|
| **高压安全** | 高压警示标识，防护罩，联锁装置，测试过程中禁止接触 |
| **急停按钮** | 设备四角各一个 |
| **PPE** | 绝缘手套、安全眼镜、劳保鞋 |
| **接地保护** | 设备可靠接地，接地电阻 ≤ 4Ω |

### 6. 设备验收标准 (FAT / SAT)

| 验收阶段 | 方法 | 合格标准 |
|----------|------|----------|
| **预验收 (FAT)** | 供应商场地使用标准定子验证 | 测试精度符合规格要求 |
| **终验收 (SAT)** | 买方产线，完成 GR&R 测试 | 连续运行 24 小时，测量一致性 GR&R ≤ 10% |

## 📄 9.2 SOP — 电机定子性能测试标准作业指导书

### 1. 准备工作与 5S 要求

| 项目 | 要求 |
|------|------|
| **PPE** | 绝缘手套、安全眼镜、劳保鞋 |
| **5S** | 工位整洁，工具定置定位 |
| **开班点检** | 检查各测试模块校准状态，使用标准电阻/电容验证精度 |
| **首件检验** | 使用标准定子（已知参数）验证设备测试精度 |

### 2. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 扫描产品条码，系统自动调用测试程序 | 条码错误，程序锁定 |
| **2. 安装** | 将定子安装到测试台，连接测试线缆 | 连接牢固，接触良好 |
| **3. 启动** | 按下启动按钮，系统自动执行测试 | 高压测试时禁止接触 |
| **4. 测试** | 系统自动执行：绝缘→耐压→匝间→电感 | 观察测试过程，无异常 |
| **5. 判定** | 系统自动判定 OK/NOK | OK 件取出；NOK 件隔离 |
| **6. 记录** | 系统自动生成测试报告 | 数据自动上传 MES |

### 3. 异常处理与升级流程 (Escalation)

| 级别 | 触发条件 | 处理措施 |
|------|----------|----------|
| **轻微报警** | 单次 NOK | 复位重测，确认定子状态 |
| **严重报警** | 连续 3次 NOK，或测试异常（击穿、冒烟） | **立即停止**，按急停，拉 Andon，通知设备工程师 + 工艺工程师 + EHS |

### 4. 首件检验规范

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验工具** | 标准定子（已知参数） |
| **检验方法** | 安装标准定子进行测试，确认设备测试精度 |
| **合格标准** | 测量值与标准值偏差在允许范围内 |
| **记录** | 填写首件检验记录表，班长签字确认 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品测试前必须扫码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与测试数据关联上传 MES |

## 📄 9.3 PFMEA — 电机定子性能测试过程失效模式与影响分析

> 基于 AIAG-VDA 统一标准格式编制

| 过程步骤 | 潜在失效模式 (FM) | 潜在失效影响 (Effect) | 严重度 (S) | 潜在失效原因 (Cause) | 频度 (O) | 现行预防控制 (PC) | 现行探测控制 (DC) | 探测度 (D) | RPN | 建议优化措施 (Action) |
|----------|-------------------|-----------------------|------------|-----------------------|----------|-------------------|-------------------|------------|-----|----------------------|
| **10. 绝缘测试** | 绝缘击穿 | 电机短路，安全事故 | 9 | 绝缘材料缺陷，潮湿污染 | 2 | 来料检验，环境控制 | **绝缘电阻实时监控**，击穿立即停止 | 2 | **36** | 引入绝缘在线监测 |
| **20. 耐压测试** | 击穿 | 电机短路，安全事故 | 9 | 绝缘厚度不足，装配损伤 | 2 | 来料检验，装配过程控制 | **漏电流实时监控**，超限立即停止 | 2 | **36** | 引入耐压在线监测 |
| **30. 匝间测试** | 匝间短路 | 性能下降，电机烧毁 | 8 | 绕组绝缘破损，漆包线质量不良 | 2 | 漆包线来料检验，绕线工艺控制 | **波形比较分析**，异常立即报警 | 3 | **48** | 引入匝间绝缘在线监测 |
| **40. 电感测试** | 电感异常 | 性能偏差，效率下降 | 6 | 匝数错误，铁芯叠压不良 | 2 | 绕线工艺控制，铁芯工艺控制 | **电感值自动判定**，超限报警 | 2 | **24** | 无需额外措施 |
