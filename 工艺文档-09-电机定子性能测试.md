## 📄 9.1 URS — 电机定子性能测试设备技术规格书

> 本规格书为招标级文件，供应商须逐条响应，任何偏离须在投标阶段书面说明。

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 电机定子综合性能测试设备 |
| **设备节拍 (C/T)** | 单件 ≤ 30 秒（含安装、测试、判定、卸载） |
| **制程能力要求** | 测量系统 GR&R %SV ≤ 10%，ndc ≥ 5 |
| **设备综合效率** | OEE ≥ 95% |
| **可靠性** | MTBF ≥ 5000h，MTTR ≤ 30min |
| **设备寿命** | ≥ 10 年（核心测试模块按 5 年校准周期） |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 | 备注 |
|------|----------|------|
| **绝缘电阻测试** | 测试电压 500V DC，测量范围 0–1000MΩ，精度 ≤ ±5% | 符合 IEC 60243 |
| **耐压测试** | 测试电压 0–5000V AC 可调，漏电流 ≤ 10mA，判定阈值可编程 | 符合 IEC 60034-1 |
| **匝间绝缘测试** | 冲击电压 0–5000V，波形采样率 ≥ 100MSa/s，比较面积差阈值可设 | Lissajous 图形分析 |
| **电感测试** | 测试频率 1kHz / 10kHz 可选，测量范围 0.1–1000mH，精度 ≤ ±1% | LCR 表分辨率 0.001mH |
| **接地电阻测试** | 测试电流 10A AC，测量范围 0–1Ω，精度 ≤ ±1% | 符合 IEC 60204-1 |

### 4. 控制策略与防错逻辑 (软件要求)

| 测试项 | 控制逻辑 | 防呆机制 |
|--------|----------|----------|
| **绝缘电阻** | 施加 500V DC，稳态 5s 后读数，判断 ≥ 100MΩ | 未达到稳态时间禁止读数 |
| **耐压测试** | 施加 1500V AC / 60s，漏电流实时监控，超 10mA 立即切断 | 电压缓升 0→1500V 时间 ≥ 2s |
| **匝间绝缘** | 冲击电压激发，双通道波形采集，面积差/频率差比较 | 无波形数据禁止判定 |
| **电感测试** | 4 线法测量，开路/短路补偿自动切换 | 测量夹具不到位禁止测试 |
| **接地电阻** | 10A AC 恒流源，电压降法测量 | 回路未闭合自动报警 |
| **自动判定** | 所有测试项独立判定，全部 OK 才放行 | 任一 NOK 件自动隔离，禁止混入 |

### 5. MES 信息化握手协议（OPC UA 强制）

| 项目 | 规格要求 |
|------|----------|
| **通信协议** | OPC UA（IEC 62541），必须支持 TSN |
| **上传字段（12 项）** | ① 产品 SN ② 测试时间 ③ 判定结果(OK/NOK) ④ 绝缘电阻(MΩ) ⑤ 耐压漏电流(mA) ⑥ 匝间绝缘波形路径 ⑦ 电感值(mH) ⑧ 接地电阻(Ω) ⑨ 各测试项子结果 ⑩ 设备 ID ⑪ 操作工 ID(RFID) ⑫ 测试次数计数 |
| **上传失败处理** | 自动重试 3 次，间隔 2s/5s/10s；3 次均失败则本地缓存 |
| **本地缓存** | SQLite 数据库，容量 ≥ 10000 条，断网恢复后自动补传 |
| **数据存储** | ≥ 100 万条记录，支持趋势分析与 SPC |
| **操作工身份** | RFID 卡认证，未刷卡禁止启动设备 |

### 6. EHS 与人体工程学（高压安全特殊要求）

| 项目 | 要求 | 依据标准 |
|------|------|----------|

| **防护罩** | 高压测试区域全封闭防护罩，透明观察窗（PC 材质，厚度 ≥ 5mm） | EN ISO 14120 |
| **联锁装置** | 防护罩开启时高压回路强制断开；关闭后延时 2s 再上电 | EN ISO 14119 |
| **PPE** | 绝缘手套（Class 0，1000V）+ 安全眼镜 + 劳保鞋（防砸防刺穿） | EN 60903 / EN 166 |
| **接地保护** | 设备可靠接地，接地电阻 ≤ 4Ω，每年复测 | IEC 60204-1 |
| **急停按钮** | 四角各 1 个 + 操作面板 1 个，共 5 个；硬线断开主电源，自锁式 | EN ISO 13850 |
| **安全光栅** | 上下料区域安装，响应时间 ≤ 10ms，光轴间距 ≤ 14mm | IEC 61496-1 |
| **双手启动** | 间距 ≥ 260mm，时间差 ≤ 0.5s，防止单手操作 | EN 574 |
| **噪音** | 设备运行噪音 ≤ 72dB(A) @1m | GBZ 2.2 |
| **上料高度** | 800 ± 50mm（可调），符合人体工程学 | EN 1005-2 |

## 📄 9.2 SOP — 电机定子性能测试标准作业指导书

> 本文件为车间受控级文件，操作工须经培训并考核合格后方可上岗。

### 1. 开班点检清单（每班必做，共 8 项）

| 序号 | 点检项目 | 方法 | 判定标准 | 不合格处理 |
|------|----------|------|----------|------------|
| 1 | 标准电阻验证精度 | 接入 100MΩ 标准电阻，测量 3 次 | 偏差 ≤ ±3% | 报修 |
| 2 | 标准电容验证精度 | 接入标准电容器，测量 3 次 | 偏差 ≤ ±3% | 报修 |
| 3 | 绝缘电阻测试仪校准 | 使用标准高压源验证 | 偏差 ≤ ±5% | 报修 |
| 4 | 耐压测试仪校准 | 使用标准高压源验证 | 偏差 ≤ ±5% | 报修 |
| 5 | 安全联锁测试 | 打开防护罩，确认高压断开 | 高压回路 100% 断开 | 禁止开机 |
| 6 | 急停按钮测试 | 逐一按下 5 个急停按钮 | 主电源立即断开 | 禁止开机 |
| 7 | 高压警示确认 | 目视检查警示标识完整 | 标识清晰无遮挡 | 粘贴补充 |
| 8 | 接地电阻确认 | 接地电阻测试仪测量 | ≤ 4Ω | 禁止开机 |

> 点检完毕后在点检表上签字确认，异常项须记录并上报班长。

### 2. 首件检验规范

| 项目 | 要求 |
|------|------|
| **触发时机** | 每班开班前 / 换型后 / 设备重启后 / 故障恢复后 |
| **检验工具** | 标准定子（已知绝缘电阻、电感、耐压参数） |
| **检验方法** | 安装标准定子，执行 3 次全项测试 |
| **合格标准** | 3 次测试结果与标准值偏差均在允许范围内，且 3 次之间一致性 ≤ ±2% |
| **记录** | 填写首件检验记录表，操作工 + 班长双签确认 |
| **不合格处理** | 禁止生产，立即通知 PE/QE 排查 |

### 3. 标准操作步骤（6 步 × 3 列）

| 步骤 | 动作描述 | CTQ / 防呆 |
|------|----------|------------|
| **1. 身份认证** | RFID 卡刷卡登录，系统校验操作工资质 | 无资质 / 未刷卡 → 禁止启动 `[插入图片：RFID刷卡位置]` |
| **2. 扫码绑定** | 扫描产品条码，系统自动调用对应测试程序 | 无码 / 重码 / 码模糊 → 程序锁定 `[插入图片：扫码枪使用]` |
| **3. 安装定位** | 将定子安装到测试台，确认定位销到位，连接测试线缆 | 定位销未到位 → 夹具报警；线缆虚接 → 接触电阻超差 `[插入图片：定子安装]` |
| **4. 安全确认** | 关闭防护罩，双手同时按下启动按钮（间距 ≥ 260mm，时间差 ≤ 0.5s） | 防护罩未关 → 无法启动；单手操作 → 无法启动 `[插入图片：双手启动]` |
| **5. 自动测试** | 系统自动执行：绝缘 → 耐压 → 匝间 → 电感 → 接地，全程约 25s | 高压测试中禁止打开防护罩（联锁）；任何异常 → 自动急停 `[插入图片：测试界面]` |
| **6. 判定卸载** | 系统显示 OK/NOK → OK 件取出放合格品区；NOK 件取出放红色隔离箱并贴标签 | NOK 件禁止混入合格品；数据自动上传 MES `[插入图片：判定界面]` |

### 4. Escalation 矩阵（4 级升级）

| 级别 | 触发条件 | 责任人 | 响应时间 | 处理措施 |
|------|----------|--------|----------|----------|
| **L1** | 单件 NOK | 操作工 | ≤ 2min | 复测 1 次确认；仍 NOK → 隔离并记录 |
| **L2** | **连续 3 件 NOK** | 班长 | ≤ 5min | **强制停机**，排查设备 / 工装 / 来料 |
| **L3** | 连续 5 件 NOK 或测试异常（击穿、冒烟） | PE / QE | ≤ 30min | 停线分析，通知设备 + 工艺 + EHS |
| **L4** | 批量 ≥ 20 件 NOK | 工程经理 | 立即 | **停线 + CAPA**，启动 8D 报告，含 EHS 通知 |

> L2 及以上升级须在 5 分钟内填写《异常升级报告》，L4 升级须同步通知工厂管理层。

## 📄 9.3 PFMEA — 电机定子性能测试过程失效模式与影响分析

> 基于 AIAG-VDA 统一标准格式编制。PC = 预防控制，DC = 探测控制。

| 步骤 | 失效模式 (FM) | 失效影响 (Effect) | S | 失效原因 (Cause) — 4M1E | O | PC（预防控制） | DC（探测控制） | D | RPN | AP | 建议行动 (Action) | 责任方 |
|------|---------------|-------------------|---|--------------------------|---|----------------|----------------|---|-----|----|-------------------|--------|
| **10 绝缘测试** | 绝缘电阻低于阈值（假 OK） | 电机运行中绝缘击穿，烧毁 / 人身伤害 | 9 | **Man**: 操作工未检查线缆连接；**Machine**: 测试仪高压输出偏低；**Material**: 绝缘材料受潮 / 含杂质；**Method**: 测试时间不足；**Environment**: 车间湿度 >80% | 2 | 来料 IQC 检验；线缆连接 SOP 标准化；车间温湿度管控 | 绝缘电阻实时监控，超下限立即报警；测试后 100% 数据校验 | 2 | **36** | L | 增加绝缘电阻测试后复测逻辑 | PE |
| **20 耐压测试** | 耐压击穿 | 电机短路，安全事故 | 9 | **Man**: 定子安装偏位导致局部电场集中；**Machine**: 耐压仪输出电压漂移；**Material**: 绝缘厚度不均 / 漆层缺陷；**Method**: 电压缓升速率过快；**Environment**: 粉尘附着 | 2 | 安装定位销防呆；耐压仪年度校准；电压缓升 ≥ 2s | 漏电流实时监控，超 10mA 立即切断；波形异常自动判定 | 2 | **36** | L | 引入局部放电监测（PD） | PE/EHS |
| **30 匝间测试** | 匝间短路（波形异常未检出） | 电机温升异常，效率下降，烧毁 | 8 | **Man**: —；**Machine**: 波形采集卡采样率不足；**Material**: 漆包线漆膜损伤 / 铜线毛刺；**Method**: 比较阈值设置过宽；**Environment**: — | 3 | 漆包线来料检验；绕线张力控制；波形比较阈值定期评审 | 双通道波形面积差 + 频率差双重判定；异常波形自动存档 | 3 | **72** | M | 提高采样率至 200MSa/s；增加 Lissajous 图形人工复判 | PE |
| **40 电感测试** | 电感值超差（假 OK） | 电机性能偏差，NVH 异常 | 6 | **Man**: —；**Machine**: LCR 表探针接触电阻大；**Material**: 匝数错误；铁芯叠压不良；**Method**: 测试频率选择不当；**Environment**: — | 2 | 绕线匝数 100% 计数验证；铁芯来料尺寸检验 | 电感值自动判定 + 超限报警；测试夹具定期维护 | 2 | **24** | L | 增加匝间电感差异率监控（相间偏差 ≤ 3%） | PE |
| **50 接地测试** | 接地电阻超标 | 设备漏电时人身触电 | 10 | **Man**: 接地线螺栓未拧紧；**Machine**: —；**Material**: 接地线截面积不足；**Method**: 测试电流选择不当；**Environment**: 接地桩腐蚀 | 1 | 接地线扭矩标准（按线径设定）；接地桩防腐处理 | 10A AC 恒流源测试；超限自动报警 | 1 | **10** | L | 年度接地电阻复测制度化 | EHS |
| **60 数据追溯** | MES 数据丢失 / 错误 | 产品召回时无法追溯，合规风险 | 8 | **Man**: 操作工未刷卡导致工号错误；**Machine**: OPC UA 通信中断；**Material**: —；**Method**: 未执行断网补传逻辑；**Environment**: 网络不稳定 | 2 | RFID 强制刷卡；OPC UA 心跳检测（5s） | 上传失败重试 3 次；本地 SQLite 缓存 ≥ 10000 条；补传后校验 | 2 | **32** | L | 定期数据完整性审计（月度） | IT/PE |
