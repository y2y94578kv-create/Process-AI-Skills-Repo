### 3. 控制策略与防错逻辑 (软件要求)

设备软件必须具备"力-位移曲线 (Force-Displacement Curve)"实时监控能力，并支持以下判定逻辑（Poka-Yoke）：

1.  **起始力监控 (Initial Force):** 压头接触工件时，力值若超出设定阈值（如 >500N），判定为"产品放反"或"异物"，设备必须立即停止并报警。
2.  **包络线监控 (Envelope/Window):** 支持设定不少于 5 个评估窗口（Window）或通道（Tunnel）。力-位移曲线必须100%穿过设定的通道，任何一点触碰边界，立即判定为 NOK 并截断压装。
3.  **最终位置与最终力 (End Position & Force):** 到达目标位移时，停止力必须在工艺公差范围内（如 8kN ± 0.5kN）。
4.  **曲线存储格式:** 压装力-位移曲线以 CSV 格式存储，文件命名规则：`{产品SN}_{时间戳}_{结果}.csv`，存储路径通过 MES 接口同步下发。
5.  **权限管理:** 设备参数分为操作级、工艺级、管理员级三级权限，关键参数修改需双人确认并留审计日志。
6.  **统计功能:** 内置 SPC 统计模块，支持实时 Cpk/Ppk 计算、控制图绘制、趋势预警。

## 📄 1.2 SOP — 伺服压装标准作业指导书 (车间受控级)

### 1. 开班点检清单 (每班开工前 ≤ 10 分钟)

| 序号 | 点检项目 | 方法 | 合格标准 | 不合格处置 |
|------|----------|------|----------|------------|
| 1 | OK 标准件压装验证 | 使用 OK 标准件连续压装 3 次 | 3 次均判定 OK (100% 通过) | 通知设备工程师排查 |
| 2 | NOK 标准件压装验证 | 使用 NOK 标准件连续压装 3 次 | 3 次均判定 NOK (100% 拦截) | 通知设备工程师排查 |
| 3 | 安全光栅响应测试 | 手指快速通过光栅区域 | 响应时间 < 20ms，设备立即急停 | 禁止开机，报修 |
| 4 | 急停按钮功能测试 | 按下每个急停按钮 | 5 个急停按钮均能切断主电源 | 禁止开机，报修 |
| 5 | 压头清洁度检查 | 目视 + 抹布擦拭 | 压头表面无油污、铁屑、异物 | 清洁后重新检查 |
| 6 | 工装定位销检查 | 目视 + 手感检查 | 定位销无磨损、无松动，工件放入顺畅 | 更换定位销 |

### 2. 首件检验规范 (三件验证法)

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后、工装更换后 |
| **检验工具** | OK 标准件 + NOK 高限件(过盈量偏大) + NOK 低限件(过盈量偏小) |
| **检验方法** | 标准件 × 3 次验证：OK 标准件 → 3 次压装 → 全部判定 OK；NOK 高限件 → 3 次压装 → 全部判定 NOK；NOK 低限件 → 3 次压装 → 全部判定 NOK |
| **判定准则** | 全部 9 次压装结果正确方可放行生产 |
| **记录** | 填写首件检验记录表，班长签字确认，拍照存档 |
| **异常处理** | 首件检验不合格时，禁止开机生产，立即通知 PE/QE 排查 |
| **保存要求** | 首件检验记录保存 ≥ 2 年，电子版同步上传 MES |

### 3. 标准操作步骤 (5 步 × 3 列)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 拿起待压装的子件和主壳体，依次在扫码枪下扫描，听见"滴"声确认 | 屏幕显示条码变绿；**防呆：** 无条码/重码/条码模糊 → 设备锁定无法启动。 `[插入图片：扫码操作示意图]` |
| **2. 上料** | 将主壳体放入底座定位销中，再将子件对准孔位轻轻放入 | **防反：** 壳体定位缺口朝向操作员；工装有防反销，放错无法放入。确认工件完全落位，无翘起。 `[插入图片：上料定位示意图]` |
| **3. 启动** | 双手离开光栅区域，同时按下左、右两侧绿色启动按钮，直到压头开始下降后方可松手 | **安全：** 两手按下时间差 ≤ 0.5s；中途手伸入光栅 → 设备报警急停。 `[插入图片：双手启动操作示意图]` |
| **4. 判定** | 观察设备屏幕显示的力-位移曲线和判定结果 | 绿色 **OK** → 压头自动抬起；红色 **NOK** → 警报灯亮起。操作工不得在压装过程中干预。 `[插入图片：OK/NOK曲线对比图]` |
| **5. 下料** | 取出压装好的组件，OK 件放入右侧蓝筐，NOK 件放入左侧红筐 | **严禁：** NOK 件混入良品筐。NOK 件立即贴红色不良品标签并记录。 `[插入图片：下料与分拣示意图]` |

### 行动优先级 (AP) 说明

| AP | RPN 范围 | 处理要求 |
|----|----------|----------|
| **H (高)** | RPN ≥ 60 或 S ≥ 8 + D ≤ 2 | 必须立即采取纠正措施，15 天内闭环 |
| **M (中)** | 40 ≤ RPN < 60 | 应制定改进计划，30 天内闭环 |
| **L (低)** | RPN < 40 | 维持现有控制，季度回顾 |

### 4M1E 根因编码说明

| 编码 | 类别 | 说明 |
|------|------|------|
| **人 (Man)** | 人员操作 | 培训不足、疲劳操作、注意力不集中、未按 SOP 执行 |
| **机 (Machine)** | 设备工装 | 设备参数漂移、工装磨损、传感器失灵、电气元件老化 |
| **料 (Material)** | 物料来料 | 来料超差、混料、表面缺陷、包装损坏 |
| **法 (Method)** | 工艺方法 | 参数设置不合理、SOP 不完善、检验标准缺失、文件版本混乱 |
| **环 (Environment)** | 环境条件 | 温湿度变化、振动、粉尘、电磁干扰、照明不足 |

### RPN 计算说明

| 参数 | 定义 | 评分标准 |
|------|------|----------|
| **严重度 (S)** | 失效对客户/后工序的影响程度 | 1-10 分，10 为最严重（安全事故/批量召回） |
| **频度 (O)** | 失效原因发生的频率 | 1-10 分，10 为极高频（每件发生） |
| **探测度 (D)** | 现行控制检出失效的能力 | 1-10 分，10 为极难检出（无任何控制） |
| **RPN** | S × O × D | 1-1000，值越高风险越大 |
