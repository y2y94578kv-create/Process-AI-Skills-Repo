# 🔧 工艺 1：伺服压装工艺 (Press-fitting)

---

## 📄 1.1 URS — 伺服压装设备技术规格书

### 1. 项目概况与总体要求

| 项目 | 说明 |
|------|------|
| **设备名称** | 精密伺服压装专机 |
| **设备节拍 (C/T)** | 单件完成时间 ≤ 12 秒（含上下料、扫码、压装、判定） |
| **制程能力要求** | 关键特性（压装到位尺寸、最终压入力）**Cpk ≥ 1.67**，测量系统分析 **GR&R ≤ 10%** |

### 2. 核心工艺技术参数 (硬件要求)

| 参数 | 规格要求 |
|------|----------|
| **压装执行机构** | 全闭环伺服压机（推荐：Kistler 奇石乐 / Promess / 太平洋） |
| **力传感器** | 高精度力传感器，额定量程 10 kN |
| **力值精度** | ≤ ±0.3% FS（全量程） |
| **位移光栅尺** | 高精度光栅尺，分辨率 0.001 mm |
| **最大行程** | 200 mm |
| **重复定位精度** | ≤ ±0.01 mm |
| **同轴度与垂直度** | 压头与底座工装的相对垂直度 ≤ 0.02 mm / 100 mm |

### 3. 控制策略与防错逻辑 (软件要求)

设备软件必须具备"力-位移曲线 (Force-Displacement Curve)"实时监控能力，并支持以下判定逻辑（Poka-Yoke）：

1.  **起始力监控 (Initial Force):** 压头接触工件时，力值若超出设定阈值（如 >500N），判定为"产品放反"或"异物"，设备必须立即停止并报警。
2.  **包络线监控 (Envelope/Window):** 支持设定不少于 5 个评估窗口（Window）或通道（Tunnel）。力-位移曲线必须100%穿过设定的通道，任何一点触碰边界，立即判定为 NOK 并截断压装。
3.  **最终位置与最终力 (End Position & Force):** 到达目标位移时，停止力必须在工艺公差范围内（如 8kN ± 0.5kN）。

### 4. 工业控制、MES与数据追溯

| 项目 | 规格要求 |
|------|----------|
| **PLC 控制系统** | 西门子 S7-1200/1500 或三菱 Q/iQ 系列 |
| **扫码追溯** | Keyence/Cognex 固定式或手持式扫码枪，无条码/重码防呆 |
| **数据上传** | OPC UA 或 TCP/IP 协议接口 |
| **数据内容** | 产品条码、最终压装力、最终位移值、OK/NOK 结果、CSV 格式曲线数据 |
| **数据存储** | ≥10万条记录 |

### 5. EHS (环境、健康与安全)

| 项目 | 要求 |
|------|------|
| **安全防护** | 操作开口处标配安全光栅（Sick / Omron），响应时间 ≤ 20ms |
| **急停功能** | 光栅触发后设备立即实施硬中断断电刹车 |
| **双手启动** | 必须配备双手启动按钮，两手按下时间差 ≤ 0.5 秒 |
| **噪音等级** | 正常运行时 ≤ 75 dB |
| **PPE** | 安全眼镜、防割手套、劳保鞋 |

### 6. 设备验收标准 (FAT / SAT)

| 验收阶段 | 方法 | 合格标准 |
|----------|------|----------|
| **预验收 (FAT)** | 供应商场地内连续压装 300 件 | 零故障，良率 ≥ 99.8% |
| **终验收 (SAT)** | 买方产线内，提供第三方力值及位移标定证书 | 完成 GR&R 测试，连续运行 24 小时验证 Cpk |

## 📄 1.2 SOP — 伺服压装标准作业指导书

### 1. 准备工作与 5S 要求

| 项目 | 要求 |
|------|------|
| **PPE** | 安全眼镜（防飞溅）、防割手套、劳保鞋 |
| **5S** | 工位整洁，工具定置定位，废料及时清理 |
| **开班点检** | 使用抹布和无水酒精清洁压头及工装底座 |
| **首件检验** | 班长使用"点检标准件（Master Part）"打件：一个模拟 OK 曲线的实心块，一个模拟过盈量过大的 NOK 块，确认设备能正确拦截 NOK 后方可开始生产 |

### 2. 标准操作步骤 (Step-by-Step)

| 步骤 | 动作描述 | 关键质量要点 (CTQ) / 防呆 |
|------|----------|--------------------------|
| **1. 扫码** | 拿起待压装的子件和主壳体，依次在扫码枪下扫描 | 听见"滴"声，屏幕显示条码变绿。若条码错误，设备锁定 |
| **2. 上料** | 将主壳体放入底座定位销中，再将子件对准孔位轻轻放入 | **注意防反：** 确认壳体的定位缺口朝向操作员方向。工装有防反销，放错无法放入 |
| **3. 启动** | 双手离开光栅区域，同时按下左、右两侧的绿色启动按钮 | 按下直到压头开始下降后方可松手。若中途手伸入光栅，设备会报警急停 |
| **4. 判定** | 观察设备屏幕显示的曲线和判定结果 | 屏幕显示绿色 **OK**，压头自动抬起；显示红色 **NOK**，警报灯亮起 |
| **5. 下料** | 取出压装好的组件。OK 件放入右侧蓝筐；NOK 件放入左侧红筐 | **严禁将 NOK 件混入良品筐** |

### 3. 异常处理与升级流程 (Escalation)

| 级别 | 触发条件 | 处理措施 |
|------|----------|----------|
| **轻微报警** | 连续 ≤ 2次 NOK | 操作工自行复位，将不良品隔离登记，重新拿取新零件进行压装 |
| **严重报警** | 连续 3次 NOK 或异常异响 | **立即停止操作**，按下急停按钮，拍下屏幕报警界面，拉响车间 Andon（暗灯）系统，或对讲机呼叫**带班班长**及**工艺工程师（PE）** |
| **禁止事项** | — | **禁止操作工擅自打开电气柜或修改压装参数** |

### 4. 首件检验规范

| 项目 | 要求 |
|------|------|
| **检验频次** | 每班开班前、换型后、设备重启后 |
| **检验工具** | OK 标准件、NOK 标准件 |
| **检验方法** | 使用标准件压装，确认设备判定结果正确 |
| **记录** | 填写首件检验记录表，班长签字确认 |

### 5. 条码追溯操作

| 项目 | 要求 |
|------|------|
| **扫码时机** | 每件产品压装前必须扫码 |
| **扫码内容** | 子件条码 + 主壳体条码 |
| **防呆措施** | 无条码、重码、条码模糊均无法启动设备 |
| **数据关联** | 系统自动将条码与压装数据关联上传 MES |

## 📄 1.3 PFMEA — 伺服压装过程失效模式与影响分析

> 基于 AIAG-VDA 统一标准格式编制

| 过程步骤 | 潜在失效模式 (FM) | 潜在失效影响 (Effect) | 严重度 (S) | 潜在失效原因 (Cause) | 频度 (O) | 现行预防控制 (PC) | 现行探测控制 (DC) | 探测度 (D) | RPN | 建议优化措施 (Action) |
|----------|-------------------|-----------------------|------------|-----------------------|----------|-------------------|-------------------|------------|-----|----------------------|
| **20. 零件放置** | 零件放反/错位 | 零件被压坏，设备压头受损受力偏载 | 7 | 定位工装磨损，防反特征不明显，员工疲劳操作 | 3 | 1. 治具设计防反销（Poka-Yoke）<br>2. 治具使用高硬度防磨材质 | 1. 压机设置"起始碰触力"监控（阈值>50N即报警） | 2 | **42** | 定期（每季度）使用三坐标校验定位治具的磨损尺寸 |
| **30. 伺服压装执行** | **压装力过大**（曲线突破包络线上限） | 壳体破裂，内应力过大导致后期使用中疲劳断裂 | 8 | 1. 子件外径超差（偏大）<br>2. 孔径超差（偏小）<br>3. 表面无润滑层/生锈卡涩 | 4 | 1. 来料尺寸抽检管控<br>2. 来料表面防锈及清洁度管控 | 1. **全闭环力-位移包络线实时监控**，触碰上限立即截断报警 | 2 | **64** | 引入 SPC 控制图，对压装最终力进行趋势监控，在触碰报警限前进行预警干预 |
| **30. 伺服压装执行** | **压装力过小**（曲线掉出包络线下限） | 配合过盈量不足，产品运行中振动导致子件脱落 | 8 | 1. 子件外径偏小<br>2. 孔径偏大<br>3. 错误使用混料零件 | 3 | 1. 供应商出厂全检关键尺寸<br>2. 产线执行严格物料隔离（防混料） | 1. **力-位移包络线实时监控**，力量不足立即判定 NOK。<br>2. 破坏性推力抽检（每班1次）。 | 2 | **48** | 要求零件供应商提供出货关键尺寸的 Cpk 报告（Cpk>1.33） |
| **30. 伺服压装执行** | **最终深度不到位**（未达到目标位移） | 后道工序装配干涉，整体尺寸超差 | 6 | 1. 孔底有异物/切屑积压<br>2. 压机行程参数被误修改 | 2 | 1. 压装前增加高压吹气清洁孔内工位<br>2. 设备参数设定 HMI 权限密码锁定 | 1. 压机伺服光栅尺进行"最终停止位置"公差窗口判定 | 2 | **24** | 无需额外措施，保持当前控制 |
