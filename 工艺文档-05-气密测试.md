### 2. 核心工艺技术参数

| 参数 | 规格要求 |
|------|----------|
| **测试压力** | 10–500 kPa 可调，分辨率 ≤ 0.1 kPa，线性度 ≤ ±0.05% FS |
| **保压时间** | 1–300 秒可调，计时精度 ≤ ±0.01s |
| **允许泄漏率** | ≤ 0.1 ml/min（精密级）/ ≤ 1 ml/min（标准级），可按产品分程序设定 |
| **测试介质** | 压缩空气（ISO 8573-1 Class 1，露点 ≤ -40°C）或氦气（纯度 ≥ 99.999%） |
| **压力传感器** | Setra / WIKA，精度 ≤ ±0.1% FS，响应 ≤ 1ms，量程覆盖 0–600 kPa |
| **流量传感器** | Brooks / Alicat，精度 ≤ ±0.5%，量程覆盖 0.001–100 ml/min |
| **密封工装** | 自动密封，密封力可控（50–500N），密封圈材质：FKM/EPDM，累计寿命计数器 |
| **压力衰减分辨率** | ≤ 0.01 kPa（精密级），≤ 0.1 kPa（标准级） |
| **温度补偿** | 自动温度补偿，补偿精度 ≤ ±0.5%，工作温度范围 15–35°C |

### 3. 控制策略与防错逻辑

| 策略 | 说明 |
|------|------|
| **压力衰减法** | 充气→稳压→保压→检测压降，判定泄漏率，主模式，适用于绝大多数产品 |
| **流量法** | 直接测量泄漏流量，适用于高精度检测或大泄漏量产品 |
| **氦检漏法（可选）** | 氦气示踪气体，质谱仪检测，灵敏度 ≤ 10⁻⁶ mbar·L/s，适用于军工/航天级产品 |
| **多腔体测试** | 同时测试多个腔体，独立判定，适用于多腔结构产品 |
| **温度补偿** | 自动温度补偿，消除环境温度影响，补偿精度 ≤ ±0.5%，实时采集环境温度 |
| **泄漏率-压降双校验** | 两种方法交叉验证，任一超限即判定 NOK，消除单一方法误判风险 |
| **保压时间-压力曲线自适应** | 根据产品特性自动匹配保压时间，压力曲线异常自动缩短/延长保压 |
| **多级判定阈值** | OK / 边界 / NOK 三级判定，边界件自动隔离复检 |

### 7. FAT/SAT 验收量化指标

#### 7.1 FAT 预验收（供应商场地）

| 验收项目 | 合格标准 | 验证方法 | 判定 |
|----------|----------|----------|------|
| **标准漏孔验证** | 泄漏率测量误差 ≤ ±5% | 高限漏孔(×1.2)测试 3 次 → 100% 报警 NOK；低限漏孔(×0.8)测试 3 次 → 100% 通过 OK | 全部通过为合格 |
| **GR&R** | %SV ≤ 10%，ndc ≥ 5 | 10 件样品 × 3 人 × 3 次，Minitab 交叉分析 | 全部通过为合格 |
| **节拍验证** | 单件 ≤ 60s | 连续生产 10 件，取平均值 | 全部通过为合格 |
| **安全功能** | 全部安全功能有效 | 安全光栅/急停/双手启动/联锁逐项测试 | 全部通过为合格 |
| **MES 通信** | 12 项字段完整上传 | 模拟 10 件测试数据上传验证 | 全部通过为合格 |
| **压力稳定性** | 稳压阶段压力波动 ≤ ±0.1% FS | 保压 60s，记录压力曲线 | 全部通过为合格 |
| **温度补偿** | 环境温度 15–35°C 范围内，泄漏率偏差 ≤ ±0.5% | 改变环境温度，验证补偿效果 | 全部通过为合格 |

#### 7.2 SAT 终验收（买方产线）

| 验收项目 | 合格标准 | 验证方法 | 判定 |
|----------|----------|----------|------|
| **Cpk** | ≥ 1.67 | 连续生产 ≥ 30 件，Minitab 计算泄漏率 Cpk | 全部通过为合格 |
| **OEE** | ≥ 95% | 可用率 × 性能率 × 良品率，连续 8h 验证 | 全部通过为合格 |
| **MTBF** | ≥ 5000h | 定时截尾法（现场运行数据统计，≥ 2000h 有效数据） | 全部通过为合格 |
| **MTTR** | ≤ 30min | 从故障停机到恢复生产的平均时间（含诊断+修复） | 全部通过为合格 |
| **连续稳定性** | 连续 24h 无故障 | 自动运行 24h，记录所有异常事件 | 全部通过为合格 |

#### 7.3 GR&R 测试执行细则

| 项目 | 要求 |
|------|------|
| **样品选择** | 10 件样品覆盖工艺规格范围（含 OK 件 + NOK 边界件 + NOK 超差件） |
| **评价人** | 3 名操作工，独立盲测，不可参考他人结果 |
| **重复次数** | 每人对每件样品测试 3 次，共 90 次测量（10×3×3） |
| **分析工具** | Minitab → 交叉 GR&R（Crossed），输出 %SV、ndc、贡献率 |
| **判定标准** | %SV ≤ 10%：可接受；10% < %SV ≤ 30%：有条件接受（需改进）；%SV > 30%：不可接受 |
| **ndc 要求** | ndc ≥ 5（可区分类别数），反映测量系统分辨能力 |
| **报告输出** | Minitab 报告 + 原始数据表，归档保存 ≥ 5 年 |

#### 7.4 OEE 计算公式与验证方法

| 指标 | 公式 | 说明 |
|------|------|------|
| **可用率** | (计划运行时间 - 停机时间) / 计划运行时间 × 100% | 停机含故障、换型、等待 |
| **性能率** | (理论节拍 × 总产出) / 实际运行时间 × 100% | 理论节拍 = 60s/件 |
| **良品率** | OK 件数 / 总测试件数 × 100% | NOK 件不含重测合格件 |
| **OEE** | 可用率 × 性能率 × 良品率 × 100% | 连续 8h 验证，≥ 95% 为合格 |

### 4. Escalation 矩阵（4 级升级）

| 级别 | 触发条件 | 责任人 | 响应时限 | 处置措施 | 记录要求 |
|------|----------|--------|----------|----------|----------|
| **L1-操作工** | 单件 NOK | 操作工 | 2 min 内 | 复位重测，确认产品密封性；连续 2 次 NOK → 升级 L2 | 记录 NOK 现象、复测结果 |
| **L2-班长** | 连续 3 件 NOK | 班长 | 5 min 响应 / 15 min 处置 | **强制停机**，拉 Andon，排查密封圈/工装/来料；填写异常报告 | Andon 记录 + 异常报告 + 临时对策 |

```

    ↓ 连续 3 件 NOK
```

| **响应记录** | Andon 拉动时间、响应时间、处置时间自动记录上传 MES |
| **停线计时** | 停线超过 15min 未恢复，自动升级至 L3 |

### 5. 换型规范（6 步，SMED 快速换型，目标 ≤ 10min）

| 步骤 | 动作 | 时间 | 责任人 | 关键点 |
|------|------|------|--------|--------|
| 1 | 停机 → 切断气源 → 卸载当前工装 | ≤ 1 min | 操作工 | 确认气压归零后方可拆卸 |
| 2 | 清洁工装台面，检查新工装外观 | ≤ 1 min | 操作工 | 无残留物、无划痕 |
| 3 | 安装新工装，紧固螺栓（扭矩 12–15 N·m） | ≤ 2 min | 操作工 | 扭矩扳手确认，对角紧固 |
| 4 | 接通气源，压力传感器清零 | ≤ 1 min | 操作工 | 零点漂移 ≤ ±0.05 kPa |
| 5 | 调用新产品测试程序，HMI 确认参数 | ≤ 2 min | 操作工 | 确认测试压力/保压时间/泄漏率阈值 |
| 6 | 首件检验（OK + NOK 高/低限各 3 次） | ≤ 3 min | 操作工 + 班长 | 9 次全部通过方可量产 |
